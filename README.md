万连和厉元朗握完手，又看了看王中秋，没有说话。
他不知道王中秋的身份，万一搞错了将会非常尴尬。
这下可把乔娜两口子彻底整蒙了。
厉县长！

县长，那可是一方父母官，戴鼎县排名第二的金交椅。

一想到自己刚才的表现，恨得差点怒扇几个大嘴巴子。连忙稳了稳神，挤出比哭还难看的笑，奔着厉元朗就要握手讨好。
却被早就弄得蒙头转向的王中秋，及时阻挡住。
